

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


## Steps

1. Create new project using
`nest new project-name`

2. Creating an authentication module
```
$ nest g module auth
$ nest g controller auth
$ nest g service auth

```

3. Create users service
```
$ nest g module users
$ nest g service users
```
